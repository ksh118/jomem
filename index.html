부적절한 언행 금지

  • 특정 커뮤니티 용어 및 말투 X
  • 부적절하지 않은 상황에서 금기 X
  • 과한 욕설 X (상황에 맞게 적절히 사용은 가능)
  • 시청자 간 반말 사용 X
  • 징징대기 X
  • 싸울 시 잘잘못 가리지 않고 싸운 사람 모두 밴

타스트리머 언급 금지

  • 타스트리머 및 타유튜버 홍보성 멘트 X
  • 방송 흐름과 상관없는 뜬금없는 언급 X

과한 친목 금지

  • 시청자끼리 서로 닉네임 언급 X
  • 사적인 대화 및 DM X
참여 규칙
※ 컨텐츠 참여 시, 방송 소리(사운드)를 반드시 들으면서 참여해야 합니다!

매너 관련

  • 게임 내에서 시청자끼리 귓속말 X
  • 채팅창에서 개인적인 대화 및 닉네임 언급 X
  • 경쟁 상황이라도 타인에게 큰 민폐가 되는 행위 X
  • 게임 시작 전 해당 게임의 룰을 반드시 숙지할 것

금지 사항

  • 방송 진행을 방해하는 트롤링 행위 X
  • 조밈 저격 (이유 없이 조밈만 따라다니는 행위) X
  • 편의성 모드(옵티파인 등) 외 비정상적 모드 및 핵 사용 즉시 영구 탈락
참여 방법
  1. 아래 버튼을 눌러 조밈 디스코드에 입장합니다.
  2. 디스코드 내 #역할 채널에서 '시참 참여하기' 역할을 받습니다.
  3. #서버주소 채널에 올라오는 주소와 마인크래프트 버전을 확인합니다.
  4. 서버 오픈 시 공지에 맞춰 접속합니다.

야생 배틀로얄

게임 방식

  • 시작 후 15분간 무적 및 파밍 시간이 주어집니다.
  • 시간 종료 후 PVP가 활성화되며 생존 싸움이 시작됩니다.
  • 자기장은 (0,0) 좌표를 중심으로 점점 줄어듭니다.
  • 마지막까지 살아남는 1인이 우승합니다.

상세 규칙

  • 모든 형태의 티밍 금지 (발각 시 즉시 처벌)
학교 술래잡기

게임 방식

  • 술래와 도망자로 나뉘어 학교 맵에서 진행됩니다.
  • 도망자는 제한 시간 동안 술래를 피해 살아남아야 합니다.
  • 맵 곳곳에 숨겨진 아이템을 활용하여 생존 확률을 높일 수 있습니다.
도시능력자

게임 방식

  • 모든 플레이어는 랜덤하게 능력을 부여받습니다.
  • 자신의 능력을 확인하고 다른 플레이어들과 경쟁하여 승리하세요.

명령어

/aw check

조밈을 잡아라!

게임 방식

  • 조밈이 도망자가 되고 나머지 시청자들이 술래가 됩니다.
  • 조밈을 가장 먼저 잡거나(타격), 조밈의 위치를 제보하여 기여한 분께 보상이 주어집니다.
캐치마인드

게임 방식

  • 출제자가 마인크래프트 블록이나 도구를 이용해 그림을 그립니다.
  • 나머지 플레이어들은 채팅창에 정답을 입력하여 맞혀야 합니다.
